**Step 1: Create Virtual Environment**  
-     python -m venv env_name


**Step 2: Activate Virtual Environment**
-     .\env_name\Scripts\activate

**Step 3: Check Python Version**
-     python --version

**Step 4: List Installed Packages**
-     pip list

**Step 5: Install Required Packages**
-     pip install pymongo

**Step 6: Save Installed Package**
-     pip list > requirement.txt

**Step 7: Install Django**
-     python -m pip install Django
      (after You are created and activated a virtual environment)

**Step 8: Save Installed Package**
-     pip list > requirement.txt

**Step 9: Uninstall a Package**
-     pip uninstall pymongo

**Step 10: Save**
-     pip list > requirement.txt

**Step 11: Deactivate Virtual Environment**
-     deactivate
