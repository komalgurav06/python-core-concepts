# Data Types / Object Type 

- Number : 1234, 3.1415, 3+4j, 0b111, Decimal(),function()
- string : 'spam', "bob's"
- List : [1, [2, 'three'], 4.5], list(range(10))
- Tuple : (1, 'spam', 4,)
- Dictionary : {'food':'spam'}, dict:{hours = 10}
- set : set('abc'), {'a','b','c'}  (unique value)
- Boolean : True, False
- None : None
